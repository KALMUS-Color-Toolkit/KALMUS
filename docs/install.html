Installation GuideÂ¶

The Kalmus package requires a python with version 3.7 and onward. The package is released on PyPI (Project Homepage). After you install the python>=3.7, you can install the kalmus using pip (recommended):

pip install kalmus

Alternatively, you could install the KALMUS locally by first cloning the GitHub repo of Kalmus (GitHub page). Then, move to the top directory of cloned KALMUS project folder and install using pip command:

pip install .

In both methods, the package's dependencies will be automatically installed. You can verify if the kalmus has been installed in your environment using pip command:

pip show kalmus
