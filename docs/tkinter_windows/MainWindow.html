class kalmus.tkinter_windows.MainWindowVersion2.MainWindow(barcode_tmp, barcode_gn, figsize=(12, 5), dpi=100)[source]¶

Bases: object

MainWindow Class. The main GUI window for user to interact with the Kalmus software. Has two displays for the barcodes and the histograms of their hue/brightness values. Has all buttons to the subwindow of the kalmus software.
